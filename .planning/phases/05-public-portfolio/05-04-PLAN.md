**Dynamic Lucide icon rendering:**
The icon name is stored lowercase in the database. To render the correct Lucide icon:

```tsx
import * as icons from "lucide-react";
import type { LucideIcon } from "lucide-react";

// Convert lowercase icon name to PascalCase for Lucide lookup
function getIconComponent(iconName: string): LucideIcon {
  // Convert "file-code" to "FileCode", "database" to "Database"
  const pascalCase = iconName
    .split("-")
    .map((word) => word.charAt(0).toUpperCase() + word.slice(1))
    .join("");

  const IconComponent = (icons as Record<string, LucideIcon>)[pascalCase];
  return IconComponent || icons.Code; // Fallback to Code icon
}
```

**Card design:**
- Compact card layout (flex items-center gap-3)
- Icon size: w-5 h-5 or w-6 h-6
- Icon color: --color-primary
- Name: text-sm or text-base, --color-text
- Background: subtle (bg-white/5 or similar translucent)
- Padding: px-4 py-3
- Rounded: rounded-lg
- Optional: subtle border (border border-white/10)

Keep it minimal - skills section will have many items, so each card should be clean.
