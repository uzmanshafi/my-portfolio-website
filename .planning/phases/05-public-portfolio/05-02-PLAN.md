1. Name: Large, bold heading (text-4xl sm:text-5xl md:text-6xl lg:text-7xl font-bold)

   - Use --color-primary for accent
3. Headline/Tagline: Body text (text-lg md:text-xl)
   - Use --color-text with slight opacity (opacity-80)

**CTAs container:**
- Flex row with gap, wrap on mobile
- margin-top for spacing (mt-8 md:mt-12)

- Primary style: bg-[var(--color-accent)] text-white

- Hover: slight brightness increase, subtle scale
- Links to #projects (anchor scroll)

- Secondary style: border-2 border-[var(--color-primary)] text-[var(--color-primary)]
- Same padding and rounding as primary
