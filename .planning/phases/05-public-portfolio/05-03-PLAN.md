  <files>src/app/components/portfolio/about-section.tsx</files>
  <action>
Create the AboutSection component displaying bio text and profile image.

**Props interface:**
```tsx
interface AboutSectionProps {
  description: string;
  imageUrl: string | null;
}
```

**Layout (desktop - lg+):**
- Two-column grid layout (grid grid-cols-1 lg:grid-cols-2 gap-12 lg:gap-16)
- Left column: Profile image (or placeholder if null)
- Right column: Section heading + bio text
- Items centered vertically (items-center)

**Layout (mobile/tablet):**
- Single column, image above text
- Image smaller on mobile (max-w-xs mx-auto)

**Section heading:**
- "About Me" or similar heading
- Style: text-3xl md:text-4xl font-bold mb-6
- Color: --color-text

**Profile image:**
- If imageUrl exists:
  - Use Next.js Image component for optimization
  - Aspect ratio: square (aspect-square) or 4:3
  - Rounded corners (rounded-2xl or rounded-3xl)
  - Object-cover for cropping
  - Add subtle shadow (shadow-2xl)
  - Consider border with --color-primary at low opacity
- If imageUrl is null:
  - Show a tasteful placeholder (gradient div with icon or initials)
  - Or hide image column entirely on desktop (single column layout)

**Bio description:**
- Render as paragraphs - split by double newlines if present
- Style: text-lg leading-relaxed
- Color: --color-text with slight opacity for softer read (opacity-90)
- Max width for line length (prose-like: max-w-prose or similar)
- Whitespace preserved for formatting (whitespace-pre-line)

**Container:**
- Max width container (max-w-6xl mx-auto)
- Horizontal padding (px-6 md:px-8)
