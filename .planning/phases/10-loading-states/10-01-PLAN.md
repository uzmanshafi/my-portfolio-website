  <files>src/app/globals.css</files>
  <action>
Extend globals.css with shimmer animation infrastructure:

1. Add to the existing `@theme` block (or create one after `@import "tailwindcss"`):
```css
@theme {
  --animate-shimmer: shimmer 2s ease-in-out infinite;
}
```

2. Add keyframes after the @theme block:
```css
@keyframes shimmer {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

3. Add `.skeleton` utility class after existing utilities:
```css
/* Skeleton loading placeholder with shimmer effect */
.skeleton {
  background: linear-gradient(
    100deg,
    var(--color-accent) 0%,
    var(--color-accent) 40%,
    color-mix(in oklch, var(--color-accent), white 30%) 50%,
    var(--color-accent) 60%,
    var(--color-accent) 100%
  );
  background-size: 200% 100%;
  background-attachment: fixed;
  opacity: 0.12;
  border-radius: 0.375rem; /* rounded-md */
}
```

4. Add reduced motion override:
```css
@media (prefers-reduced-motion: reduce) {
  .skeleton {
    animation: none !important;
  }
}
```

Key details:
- Use `background-attachment: fixed` for synchronized shimmer across all elements
- Use accent color (#6655b8) at ~12% opacity
- Shimmer highlight uses `color-mix()` to blend with white
- 2-second animation duration for calm, premium feel
- Diagonal gradient (100deg) for iOS-style shimmer wave
